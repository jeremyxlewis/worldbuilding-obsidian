# D&D Systems Reference

Mechanical reference for D&D 5e worldbuilding.

## Stat Block Format

Use the Fantasy Statblocks plugin format:

```statblock
name: Creature Name
size: Medium
type: humanoid
alignment: lawful good
ac: 16
hp: 75
hit_dice: 10d8
speed: 30 ft.
stats: [16, 12, 14, 10, 13, 8]
skillsaves: [Athletics, Perception]
senses: darkvision 60 ft., passive Perception 13
languages: Common, Dwarvish
cr: 3
traits:
  - name: Stone Cunning
    desc: Advantage on Intelligence (History) related to stonework.
actions:
  - name: Battleaxe
    desc: Melee Weapon Attack: +6 to hit, reach 5 ft., one target. Hit: 7 (1d8 + 3) slashing damage.
```

## Condition Reference

| Condition | Effect |
|-----------|--------|
| Blinded | Can't see, auto-fail sight checks, attacks have disadvantage |
| Charmed | Can't attack charmer, charmer has advantage on social checks |
| Deafened | Can't hear, auto-fail hearing checks |
| Frightened | Disadvantage on ability checks/attacks while source in sight |
| Grappled | Speed 0, ends if grappler incapacitated |
| Incapacitated | Can't take actions or reactions |
| Invisible | Heavily obscured, advantage on attacks, attacks against have disadvantage |
| Paralyzed | Incapacitated, auto-fail Str/Dex saves, melee hits are crits |
| Petrified | Transformed to stone, weight ×10, immune to all damage |
| Poisoned | Disadvantage on attack rolls and ability checks |
| Prone | Can only crawl, disadvantage on attacks, melee has advantage |
| Restrained | Speed 0, attacks have disadvantage, attacks against have advantage |
| Stunned | Incapacitated, auto-fail Str/Dex saves |
| Unconscious | Incapacitated, drop items, auto-fail Str/Dex saves, melee hits are crits |
